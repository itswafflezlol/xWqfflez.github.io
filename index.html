Who I am

wafflez
aka ilykeelon · itswafflez_ on Discord

yo, im waffles! also known as ilykeelon. i manage and own Minecraft servers and networks. been doing it for over 6 years and it's only gotten more interesting as time goes on.

what started as just playing on servers turned into owning them, then managing them, then helping other people run theirs. one thing led to another and now it's basically what i do.

Network management

running a network means a lot of moving parts. proxy config, server interconnection, load balancing, keeping everything stable while players are online. velocity, bungeecord, game server routing — i handle all of it.

network-level plugin management means making sure nothing conflicts, performance stays solid across the whole thing, and every server talks to each other properly.

Plugin config is an actual skill

anyone can install a plugin. getting it to work the way your server actually needs it, not conflicting with anything, optimized for your use case, that's where most people get stuck.

i've worked across every plugin category. economy, combat, permissions, events, chat, building. i know how to get them working together and staying that way.

Development

Skript is my main tool for Minecraft-specific work. when plugins can't do what you need, scripting fills the gap. i've built everything from simple chat systems to full event mechanics with custom bosses, rewards, and countdowns.

i also know HTML, and i'm working on Java, that's where i want to take things next, proper plugin development from scratch.
