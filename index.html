Introduction

yo, im waffles — also known as ilykeelon. i manage and own Minecraft servers and networks, and i help people all around with their server needs. been at this for over 2 years and honestly its just gotten more fun as time goes on.

what started as just playing on servers eventually turned into owning them, and then managing them, and then helping other people run theirs. one thing led to another and now its basically what i do.

What I Do as Network Manager

running a Minecraft network means handling a lot of moving parts — proxy configuration, server interconnection, load balancing between game modes, keeping everything online and stable. i handle all of that.

whether its setting up Velocity or BungeeCord proxies, configuring game server connections, managing player routing between servers, or troubleshooting why something isnt communicating properly — that falls under my wheelhouse.

i also deal with network-level plugin management: making sure the right plugins are on the right servers, that they dont conflict with each other, and that performance stays solid across the whole network.

Systems & Plugin Configuration

my main specialty is plugin configuration. ive worked with dozens of different plugins across all categories — economy, combat, building, events, chat, permissions, anti-cheat, and more. i know how to get them working together properly and optimized for your specific server type.

configuration is an underrated skill. anyone can install a plugin. actually making it work the way your server needs, not conflicting with anything else, and running efficiently — thats where the real work is.

i also write custom Skript code when plugins dont do exactly what you need. sometimes you need a specific mechanic that doesnt exist as a plugin — thats where scripting fills the gap.

Development Side

on the code side, Skript is my main language for Minecraft-specific work. its built for it and when you know it well, you can build pretty much any gameplay mechanic you need. ive made everything from simple chat formatting scripts to full event systems with custom bosses, rewards, and countdowns.

i also know HTML for web stuff, and im learning Java — slowly but getting there. Java opens up proper plugin development and thats where i want to take things further down the line.
